# Графическая оболочка для zapret-discord-youtube

Данное приложение представляет собой графический интерфейс (GUI) на базе Avalonia UI (.NET) для упрощения работы с утилитами обхода DPI из репозитория [flowseal/zapret-discord-youtube](https://github.com/flowseal/zapret-discord-youtube). Оно не содержит самих исполняемых файлов обхода, а только предоставляет обёртку для их запуска и настройки.

---

## Требования

- Операционная система: Windows /10 (x64). Для других ОС требуется самостоятельная сборка.
- Права администратора (для изменения сетевых настроек).
- Файлы из репозитория [flowseal/zapret-discord-youtube](https://github.com/flowseal/zapret-discord-youtube) должны быть загружены отдельно.

---

## Установка

1. Скачайте последний релиз из раздела Releases данного репозитория.
2. Распакуйте архив в любую папку.
3. Запустите `ZapretUI.exe` в обычном режиме (не от администратора).

---

## Использование

1. При первом запуске в настройкках укажите путь к корневой папке, содержащей файлы и папки `zapret-discord-youtube`.
2. Нажмите загрузить стратегии.
3. Выберите стратегию для загрузки.
4. Нажмите кнопку «Запуск» — приложение выполнит команду запуска утилит с выбранными параметрами.
5. Для остановки обхода закройте приложение.

Все настройки сохраняются в локальном конфигурационном файле.
